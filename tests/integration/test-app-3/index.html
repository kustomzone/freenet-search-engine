Freenet Chess Arena

Play chess with anyone on the Freenet network. Games are stored as contract states with cryptographic move verification, making cheating impossible. Ranked ELO matchmaking pairs you with opponents of similar skill.

Features

  • Peer-to-peer games with no central server
  • Cryptographic move verification prevents cheating
  • ELO-based ranked matchmaking
  • Game history stored as Freenet contracts
  • Weekly tournaments with community brackets

How It Works

Each game is a Freenet contract where both players submit signed moves. The contract validates move legality and updates the board state. Game results are aggregated into a ranking contract that tracks ELO scores across all players.
