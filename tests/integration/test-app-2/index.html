CryptoNotes

A fully encrypted personal notebook that stores your notes securely on the Freenet network. All encryption happens client-side using AES-256-GCM. Your notes are never visible to anyone but you, not even the nodes storing the encrypted data.

Start typing your encrypted notes here...

How It Works

CryptoNotes uses a Freenet delegate to manage your encryption keys locally. Each note is encrypted before being stored as a contract state update. The decentralized storage ensures your notes are always available while the end-to-end encryption guarantees only you can read them.

Security Features

  • AES-256-GCM client-side encryption
  • Ed25519 key management via Freenet delegates
  • Zero-knowledge note storage
  • Automatic key rotation
